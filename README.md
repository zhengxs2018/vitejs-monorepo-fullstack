一个完整的基于 [vue 3.x][vue3] 的全栈项目。

## 目录结构

```bash
Project/
├── .vscode/
├── apps/                    --- 独立应用
|   ├── api/
|   └── web/
├── libs/                     --- 公共模块
├── lerna.json
├── tsconfig.base.json
├── tsconfig.json
└── vscode.code-workspace
```

## 快速启动

注意启动项目前，必须先全局安装 `yarn` 和 `lerna` 模块。

```shell
# 安装依赖
$ yarn bootstrap

# 启动工程
$ yarn start
```
